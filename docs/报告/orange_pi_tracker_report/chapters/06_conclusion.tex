\chapter{总结}

本阶段项目已经完成两自由度云台目标跟踪系统的主要工程闭环。硬件方面，OrangePi、USB 摄像头、PCA9685 和双舵机云台已经完成连接与验证；软件方面，系统实现了基于 OpenCV 的目标检测、基于状态机的跟踪流程、基于 PCA9685 的舵机控制、运行日志记录和网页监控模式。测试结果表明，系统能够读取摄像头画面、识别红色高对比目标并驱动云台进行跟随，满足当前阶段的基础演示目标。

同时，项目已经为后续加分项打下基础：C/C++ 形态学处理模块已经实现，网页监控模式已经跑通，后续可继续完成 wheel 构建、性能对比、日志图表和网页状态面板。总体来看，本项目已经从硬件打通阶段进入稳定性优化和实验数据完善阶段。后续工作应重点围绕“稳定演示、量化评估、加分项数据化”展开。
