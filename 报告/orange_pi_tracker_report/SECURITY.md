# 安全政策 | Security Policy

## 报告安全漏洞 | Reporting a Vulnerability

本项目是一个 LaTeX 模板，不涉及网络服务或用户数据处理。如果您发现了安全相关的问题（例如 CI 配置泄露、依赖风险等），请通过 GitHub [Security Advisories](https://github.com/TJ-CSCCG/TongjiThesis/security/advisories/new) 报告。

This project is a LaTeX template and does not involve network services or user data processing. If you discover a security-related issue (e.g., CI configuration leak, dependency risk), please report it via GitHub [Security Advisories](https://github.com/TJ-CSCCG/TongjiThesis/security/advisories/new).
